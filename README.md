# TGProxy CLI

Легковесная и быстрая утилита для установки и управления собственным MTProto Proxy сервером для Telegram. Базируется на современном и надежном движке `mtg`. Поддерживает Ubuntu и Debian.

## Преимущества
* **Быстрая установка** — скрипт скачает, настроит и запустит всё сам за несколько секунд.
* **Авто-генерация секретов** — безопасный ключ (secret) с Fake-TLS создаётся автоматически.
* **Удобное управление** — просмотр ссылки, рестарт и проверка статуса через простые команды.
* **Без лишнего кода** — оставлен только функционал для MTProto, ничего лишнего.

---

## Установка утилиты

Установите `tgproxy` CLI на свой сервер всего одной командой:

```bash
curl -fsSL https://raw.githubusercontent.com/dntflp/cli/main/install.sh | sudo bash
```

*Скрипт загрузит необходимые файлы, создаст директорию `/opt/tgproxy` и подготовит базовый конфигурационный файл.*

---

## Запуск и настройка прокси

После установки утилиты запустите конфигурацию самого прокси-сервера:

```bash
sudo tgproxy install
```

Эта команда скачает актуальную версию `mtg`, сгенерирует MTProto-секрет (Fake-TLS маскируется под `google.com`), и запустит systemd-сервис.

---

## Команды TGProxy

* `tgproxy install` — Установить и настроить MTProto Proxy (скачать бинарник, создать конфиг, поднять сервис).
* `tgproxy link` — Показать вашу личную `tg://` ссылку для подключения к прокси.
* `tgproxy status` — Проверить статус работы сервиса и просмотреть конфигурацию.
* `tgproxy restart` — Перезапустить системный сервис MTProto.
* `tgproxy help` — Показать список доступных команд.

---

## Пример использования

```bash
# 1. Устанавливаем CLI
curl -fsSL https://raw.githubusercontent.com/dntflp/cli/main/install.sh | sudo bash

# 2. Поднимаем прокси
sudo tgproxy install

# 3. Получаем ссылку для Telegram
tgproxy link
```

## Конфигурация
При необходимости вы можете вручную изменить порт и домен для Fake-TLS. Конфигурация находится по пути: `/etc/tgproxy/config.env`.

После изменения файла настроек не забудьте перезапустить прокси:
```bash
sudo tgproxy restart
```
